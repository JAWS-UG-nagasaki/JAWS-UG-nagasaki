コミュニティについて

JAWS-UG（Japan AWS User Group）長崎は、全国の AWS ユーザーコミュニティの長崎支部です。長崎のエンジニアのための「ナイトハーバーハブ」として、知識を共有し、技術的な課題に取り組み、つながりを構築する場所です。

経験豊富なアーキテクトであっても、S3 や Lambda の旅を始めたばかりであっても、クラウドでの構築への「静かな情熱」をかき立てるように設計されたセッションです。
